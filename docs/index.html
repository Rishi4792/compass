The engine

Loops and graphs, with real edges.

The architecture serious builders are converging on — and one you can install today.

It's a graph

The stages are nodes; the edges are exit-code scripts. The build advances only when a gate returns zero — never because a model decided it "looked done." A state machine doesn't verify; this does.

And it's loops

Adversarial reviews repeat until two clean rounds; a post-ship critic runs on the live system until it finds nothing. Every loop is bounded — and when it runs itself, by a budget it can't exceed.

contract → review → plan → review → build → review → ship ↻ post-ship critic

Every arrow is an exit-code gate — never a model deciding it “looked done.” Nothing ships on a claim.
